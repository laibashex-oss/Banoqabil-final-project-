# BCF Foundation School Website
**Bright Children Foundation**

**Project Name:** BCF Bright Children Foundation Official Website 
**BanoQabil Final Project**

### 📖 About
This is the official website for BCF Bright Children Foundation School. The website is fully responsive and includes all essential school features. It was developed as my final project for BanoQabil.

### ✨ Key Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Hamburger Menu**: Fully functional mobile navigation 
- **Admission Form**: Online form for student admissions
- **Donation Form**: Secure donation portal for supporters
- **Complete Information**: School details, programs, and contact info

### 🛠️ Tech Stack Used
- HTML5
- CSS3
- JavaScript
